# 営業KPI管理システム

社員一人一人の営業数値管理、KPI目標設定、日次振り返りを一元管理できるWebアプリケーションです。社員は個人の目標と実績を管理し、管理者は全社員のデータを閲覧・分析できます。

## 🎯 プロジェクトの目的

- 社員が自身の営業活動を記録・振り返り、KPIを管理
- 管理者が社員の活動状況と達成状況をリアルタイムで把握
- データドリブンな営業活動の推進とチームパフォーマンスの可視化

## ✅ 実装済み機能

### 社員機能
1. **ログイン機能**
   - メールアドレスとパスワードによる認証
   - ログイン状態の永続化（localStorage使用）

2. **KPI目標設定**
   - 月次KPI目標の登録・更新
   - 売上目標、訪問件数、契約件数、架電件数の管理
   - 過去のKPI目標一覧表示

3. **営業数値入力**
   - 日次の営業実績入力（売上、訪問、契約、架電）
   - 今月の営業数値一覧表示
   - 同じ日付のデータは自動的に更新

4. **日次振り返り**
   - 本日の成果の記録
   - 課題・反省点の記録
   - 明日の予定の記録
   - 最近10件の振り返り表示

5. **ダッシュボード**
   - 今月のKPI達成率をビジュアル表示
   - 目標と実績の比較
   - 4つの指標（売上、訪問、契約、架電）の進捗状況

### 管理者機能
1. **社員フィルター機能**
   - 特定社員のデータ表示
   - 対象月でのフィルタリング

2. **KPI目標閲覧**
   - 全社員のKPI目標一覧
   - 月別・社員別でのフィルタリング

3. **営業数値閲覧**
   - 全社員の営業実績一覧
   - 日次データの確認

4. **振り返り閲覧**
   - 全社員の日次振り返り内容の確認
   - 社員の活動内容と課題の把握

5. **全体概要ダッシュボード**
   - 社員ごとの総合達成率表示
   - 4指標の達成率を一覧表示
   - 色分けによるパフォーマンス可視化（緑：達成、黄：70%以上、赤：70%未満）

## 🗄️ データモデル

### 1. employees（社員テーブル）
| フィールド | 型 | 説明 |
|----------|-----|------|
| id | text | 社員ID（ユニークID） |
| employee_id | text | 社員番号 |
| name | text | 社員名 |
| email | text | メールアドレス |
| password | text | パスワード |
| role | text | 役割（employee/admin） |
| department | text | 部署 |

### 2. kpi_targets（KPI目標テーブル）
| フィールド | 型 | 説明 |
|----------|-----|------|
| id | text | レコードID |
| employee_id | text | 社員ID |
| month | text | 対象月（YYYY-MM） |
| sales_target | number | 売上目標（円） |
| visit_target | number | 訪問件数目標 |
| contract_target | number | 契約件数目標 |
| call_target | number | 架電件数目標 |

### 3. sales_records（営業数値テーブル）
| フィールド | 型 | 説明 |
|----------|-----|------|
| id | text | レコードID |
| employee_id | text | 社員ID |
| date | text | 日付（YYYY-MM-DD） |
| sales_amount | number | 売上金額（円） |
| visits | number | 訪問件数 |
| contracts | number | 契約件数 |
| calls | number | 架電件数 |

### 4. daily_reviews（日次振り返りテーブル）
| フィールド | 型 | 説明 |
|----------|-----|------|
| id | text | レコードID |
| employee_id | text | 社員ID |
| date | text | 日付（YYYY-MM-DD） |
| achievements | rich_text | 本日の成果 |
| challenges | rich_text | 課題・反省点 |
| tomorrow_plan | rich_text | 明日の予定 |

## 🚀 使い方

### 社員の場合
1. ログイン画面で社員アカウントでログイン
2. **KPI目標設定タブ**で今月の目標を設定
3. **営業数値入力タブ**で日々の実績を入力
4. **日次振り返りタブ**で毎日の振り返りを記録
5. **ダッシュボードタブ**で進捗を確認

### 管理者の場合
1. ログイン画面で管理者アカウントでログイン
2. フィルター機能で確認したい社員・月を選択
3. 各タブで社員のデータを確認
4. **全体概要タブ**でチーム全体の状況を把握
